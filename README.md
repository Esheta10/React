# ⚛️ React — Learning & Practice

A hands-on collection of React concepts implemented from scratch — covering everything from the basics to advanced patterns.

---

## 📚 Topics Covered

- **Core Fundamentals** — JSX, components, props, rendering
- **State Management** — `useState`, state lifting, controlled components
- **Side Effects** — `useEffect`, cleanup, dependency array
- **Refs** — `useRef`, DOM access, persisting values
- **Context API** — `useContext`, global state without prop drilling
- **Performance** — `useMemo`, `useCallback`, React.memo
- **Routing** — React Router, nested routes, dynamic routes, protected routes
- **Forms** — controlled inputs, validation, form submission
- **Custom Hooks** — building and reusing your own hooks

---

## 🚀 Getting Started

```bash
# Clone the repo
git clone git@github.com:Esheta10/React.git

# Move into the project
cd React

# Install dependencies
npm install

# Start the dev server
npm start
```

App runs at `http://localhost:5173`
